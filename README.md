# llm-wiki — плагин для Claude Code

Персональная вики «второй мозг», управляемая Claude Code. Вдохновлено [подходом Андрея Карпатого](https://x.com/karpathy/status/1786228517545320794) к базам знаний, поддерживаемым LLM.

Загрузи любой источник (URL, файл, текст) → Claude синтезирует структурированные страницы знаний → Obsidian-хранилище с backlinks.

## Команды

| Команда | Описание |
|---------|----------|
| `/wiki-init` | Инициализация структуры папок и конфига Obsidian |
| `/wiki-ingest <источник>` | Загрузить URL, путь к файлу или текст в вики |
| `/wiki-query <вопрос>` | Ответить на вопрос, используя базу знаний вики |
| `/wiki-lint` | Проверка здоровья: починить индекс, frontmatter, найти orphans и битые ссылки |

## Установка

```bash
claude plugin install github:tequila99/llm-wiki-plugin
```

## Настройка

1. Создай пустую директорию для вики и открой Claude Code внутри неё.

2. Скопируй `CLAUDE.md` из этого плагина в корень вики — это схема, которая управляет тем, как Claude пишет и организует страницы. **Отредактируй её под свой язык и предпочтения** перед началом работы.

3. Выполни `/wiki-init` для создания структуры папок (`raw/`, `wiki/`) и конфига Obsidian.

4. Открой корень вики в Obsidian как хранилище (File → Open folder as vault).

## Как это работает

```
raw/articles/    ← веб-клипы, статьи, блог-посты
raw/books/       ← заметки по книгам
raw/misc/        ← ручные заметки, всё остальное
raw/             ← неотсортированные статьи для wiki
wiki/            ← синтезированные страницы знаний (Obsidian vault)
wiki/index.md    ← каталог страниц
wiki/log.md      ← аудит-журнал всех операций
```

**Файлы в `raw/` иммутабельны** — Claude никогда их не изменяет. Весь синтез происходит в `wiki/`.

Каждая wiki-страница — это Obsidian-совместимый markdown с frontmatter (`title`, `tags`, `created`, `updated`, `sources`) и `[[wikilinks]]` между связанными страницами.

## CLAUDE.md

Файл `CLAUDE.md` в корне вики определяет:
- Язык wiki-страниц (по умолчанию: русский — измени под свой язык)
- Структуру папок и соглашения об именовании
- Схему frontmatter
- Правила backlink-аудита

Этот файл загружается Claude Code автоматически как инструкции проекта. Настраивай свободно.
